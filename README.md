# Cápsula de Estudos

A Cápsula de Estudos é um sistema web desenvolvido para ajudar estudantes a organizar matérias, tópicos e revisões de forma prática em um só lugar. O projeto permite que cada usuário crie sua própria conta, cadastre as matérias que está estudando, adicione tópicos com resumos, nível de dificuldade e data de revisão, além de acompanhar o progresso por meio de um dashboard com indicadores gerais.

O sistema foi desenvolvido como projeto de prática em desenvolvimento de sistemas, com foco em aplicar conceitos de back-end, front-end e banco de dados em uma aplicação real. Entre as principais funcionalidades estão o cadastro e login de usuários com senha protegida por hash, o gerenciamento completo de matérias e tópicos (criar, editar e excluir), a marcação de tópicos como pendentes ou concluídos e a visualização de informações importantes, como total de matérias, total de tópicos, pendências e próxima revisão agendada.

## Tecnologias utilizadas

* Python
* Flask
* SQLite
* HTML
* CSS
* JavaScript
* Gunicorn

## Objetivo do projeto

O principal objetivo da Cápsula de Estudos foi praticar a construção de um sistema web completo, trabalhando autenticação, CRUD, organização de rotas, persistência de dados e estruturação de interface, transformando conceitos estudados em um projeto funcional de portfólio.
